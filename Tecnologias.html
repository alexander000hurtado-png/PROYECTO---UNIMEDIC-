Las Tecnologias Utilizadas

El proyecto consiste en el desarrollo de un sitio web institucional para Unimedic IPS, una institución prestadora de servicios de salud. La página está diseñada con un enfoque en la usabilidad, accesibilidad y profesionalismo, de manera que transmita confianza a los pacientes y usuarios interesados en los servicios de la entidad. El sitio está compuesto por múltiples secciones interconectadas mediante un menú de navegación principal, asegurando que la información sea de fácil acceso. Cada sección refleja la identidad, misión y visión de la IPS, a la vez que proporciona herramientas útiles como formularios de contacto, enlaces a redes sociales y presentación de servicios médicos..

Caracteristicas

1. Diseño responsivo El sitio utiliza HTML5, CSS3 y puede complementarse con Bootstrap para garantizar que sea accesible desde computadoras, tabletas y dispositivos móviles.

2. Páginas principales Inicio: presentación general de la IPS, su filosofía y compromiso con los pacientes. Misión y Visión: comunican los objetivos estratégicos, valores y propósitos institucionales. Valores: lista de principios que guían la práctica médica y el trato al paciente. Servicios: muestra los servicios médicos disponibles, como consultas generales, especializadas, diagnóstico, urgencias, entre otros. Quiénes Somos: espacio para presentar la historia de la IPS, su trayectoria y personal médico. Puede incluir imágenes o videos institucionales. Contacto: formulario para enviar consultas, dudas o solicitudes directamente desde la web, además de información de contacto como teléfono, dirección y correo.

3. Elementos adicionales Encabezado con logotipo y menú de navegación en todas las páginas para una navegación uniforme. Pie de página con redes sociales (Facebook, Twitter, Instagram, WhatsApp) que permiten interactuar con la comunidad y mejorar la presencia digital. Carrusel de imágenes (opcional) para resaltar servicios, instalaciones o campañas de prevención. Formulario de contacto para recibir solicitudes de pacientes de manera rápida.

4. Estilos y diseño unificado Todas las páginas comparten un archivo CSS común (estilos.css), lo que garantiza consistencia visual en colores, tipografías, botones, formularios y estructura general.

5. Escalabilidad El proyecto está pensado para expandirse fácilmente, pudiendo añadir: Agenda de citas en línea. Perfiles de médicos. Información sobre medicamentos y tratamientos. Intranet o panel para pacientes registrad

Tecnología html

HTML5 es el lenguaje estándar de marcado para la web y constituye la base estructural del proyecto. Con él se definieron los diferentes elementos que forman la página: encabezados, párrafos, menús de navegación, listas, formularios, imágenes y enlaces. Entre sus principales características en este proyecto: Se usaron etiquetas semánticas como = (header),(main),(footer), que permiten organizar mejor el contenido. Se implementó la etiqueta (form) con campos de entrada para crear el formulario de contacto. Se añadieron imágenes y videos con etiquetas como = (img) y (video). Permite que los navegadores interpreten el contenido de manera compatible y accesible, tanto en computadoras como en dispositivos móviles. HTML5 aporta la estructura fundamental sobre la cual se apoyan las demás tecnologías.

Tecnología CSS3

Define el diseño visual del sitio web. Se utiliza para aplicar colores, fuentes, márgenes, tamaños, estilos de botones, distribución de elementos y adaptación a diferentes pantallas (responsive design). En el proyecto, todos los estilos están centralizados en un archivo común (Estilos.css) para mantener la consistencia visual en todas las páginas. Ejemplos: Fondos de colores. Tamaño de íconos de redes sociales. Diseño de encabezado, pie de página y formularios

Tecnología JavaScript

Lenguaje de programación que añade interactividad al sitio web. En el proyecto, se utiliza para: Mostrar dinámicamente el año actual en el pie de página (document.getElementById('year').textContent = new Date().getFullYear()). Validar y responder al formulario de contacto con alertas de confirmación. Inicializar componentes como el carrusel de imágenes de Bootstrap o Swiper

Tecnología Swiper.js

Librería especializada en carouseles interactivos y sliders de imágenes. Permite crear presentaciones dinámicas con efectos como coverflow, autoplay o paginación. Se utiliza en el proyecto como demo para un slider de imágenes.

Tecnología Font Awesome (con CDN)

Librería de íconos que permite integrar fácilmente gráficos como los de redes sociales. Se utiliza con etiquetas para mostrar íconos vectoriales que se pueden escalar y personalizar en color y tamaño. En tu proyecto se usa en el footer para incluir enlaces a: Facebook Instagram WhatsApp

Tecnología Bootstrap (con CDN)

Es un framework de CSS y JavaScript que facilita la creación de páginas modernas y responsivas. Permite integrar componentes predefinidos como formularios estilizados, menús, botones, cuadrículas (grid system) y carouseles de imágenes sin necesidad de mucho código manual. En el proyecto se ha utilizado para: Menú de navegación adaptativo. Formularios de contacto con estilos profesionales. Carrusel de imágenes para resaltar servicios e instalaciones.

Tecnología Visual Studio Code (VS Code)

Editor de código utilizado para escribir y organizar los archivos del proyecto. Facilita la integración de extensiones para HTML, CSS, JavaScript y live server (para probar el sitio en el navegador en tiempo real).
