HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. No es un lenguaje de programación, sino un sistema de etiquetas que permite organizar el contenido en elementos como textos, enlaces, imágenes, formularios y tablas. Su función principal es definir la estructura semántica de un sitio web, indicando al navegador cómo debe mostrarse la información. Desde su versión más reciente, HTML5, incorpora etiquetas modernas que mejoran la accesibilidad, integran contenido multimedia sin necesidad de complementos y permiten desarrollar aplicaciones web más dinámicas. En el proyecto, HTML se utilizó para dar forma a la página, creando el encabezado, el menú de navegación, el formulario de contacto y la organización general del sitio.

Definió la estructura general (head, body, header, section). Organizó la información en encabezados, párrafos, enlaces y formularios. Permitió crear un menú de navegación funcional y un formulario interactivo para contacto. Se aplicaron etiquetas semánticas que mejoran la accesibilidad y la organización lógica del contenido.

CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo en cascada que controla la presentación visual de los documentos HTML, separando la estructura del contenido de su apariencia. Permite definir colores, tipografías, tamaños, márgenes, bordes, fondos y la disposición de los elementos en la página, además de incorporar animaciones y transiciones que enriquecen la experiencia del usuario. Gracias a sus capacidades de diseño responsivo, CSS asegura que los sitios web se adapten a diferentes dispositivos como computadoras, tablets y smartphones. Sus reglas se aplican siguiendo una jerarquía en cascada y pueden escribirse en línea, de forma interna o en archivos externos reutilizables.

En el proyecto, CSS se utilizó principalmente a través del framework Bootstrap 5, el cual ya incluye un conjunto de estilos predefinidos: Se aplicaron estilos al menú de navegación (navbar) para que se viera moderno y responsivo. Se dieron estilos uniformes a los campos del formulario de contacto, como bordes, colores y tamaños de entrada. El botón de envío se personalizó con clases de Bootstrap (btn btn-primary) para destacar visualmente.
